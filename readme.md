# Johann Stuckenbruck
## Public-facing site

------

## Requirements

 - API (contact [admin](mailto:info@wrweb.dev) if unsure how to proceed)
 - Node v^12
 - AWS S3 Storage Bucket (named in .env) with following folders:
   - content
   - media
   - scaled-images
 - Sendgrid account (linked in .env)

## Install

 - `npm -i`

## Run
 - `npm run dev`
